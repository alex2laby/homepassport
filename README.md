# Home Passport

Home Passport est un concept de dossier immobilier numérique destiné à centraliser, fiabiliser et valoriser les informations d'un logement tout au long de son cycle de vie.

Ce dépôt contient désormais une vitrine statique pour **HomeBook AI**, l'agent qui accompagne les propriétaires dans la préparation d'un dossier de vente complet et la génération d'un **Home Book** professionnel.

## Site vitrine

Ouvrez `index.html` dans un navigateur ou lancez un serveur local :

```bash
python3 -m http.server 4173
```

Puis visitez <http://localhost:4173>.

## Concept produit

La vraie valeur de HomeBook AI est l'agent qui :

- accompagne le vendeur question par question ;
- qualifie le bien et son contexte ;
- détecte les documents obligatoires ou manquants ;
- vérifie les diagnostics et pièces du dossier ;
- génère une fiche descriptive et un Home Book prêt à partager.
