# Home Passport

Home Passport est un concept de dossier immobilier numérique destiné à centraliser, fiabiliser et valoriser les informations d'un logement tout au long de son cycle de vie.

Ce dépôt documente les parcours produit et les fonctionnalités clés permettant de générer automatiquement un **Home Book** professionnel lors de la mise en vente d'un bien.
